## 学习内容
### CV与边缘检测

机器视觉CV问题主要有图片分类、物体检测与神经风格转换。对于这种问题，一般输入层维度很大，假如使用传统的神经网络模型，权重W就会很庞大，导致模型复杂，容易过拟合；并且内存与时间要求比较大。这便引出了CNN卷积神经网络的优势。

CV领域有一个重要的问题便是边缘检测，比如垂直边缘和水平边缘。其主要应用滤波器卷积计算来实现目的，比如垂直边缘检测使用以下示例矩阵进行卷积运算：

```
1  0  -1
1  0  -1
1  0  -1
```

在深度学习，检测图片的边缘特征也是采用滤波器，将滤波器中的全部值看作是参数，主要目的就是反复迭代计算这些参数，类似于计算W。当滤波器确定下来后，整个CNN网络也就可以实现边缘检测；

### Padding扩展

我们发现，如果在初始数据上面直接与滤波器进行卷积运算，尺寸会逐渐变小，并且丢失边缘信息。为此便有了 **Padding技术，也就是对原始尺寸进行扩展，多于区域补零。** 实际上，对于卷积运算过后的尺寸大小由公式，这个在讲完后面的步长stride之后一并给出。

### 多通道卷积

对于三通道的RGB图片，对应的滤波器也是三通道。计算过程是每个对应通道分别计算，然后将所有通道相加得到结果。不同通道间的滤波器参数可以不同。

为了实现多个卷积运算，可以增加更多的滤波器组，也就是后面要介绍的参数`nc`。比如输入图片的尺寸为`n x n x nc`，filter尺寸为`f x f x nc`，则卷积后的图片尺寸为`(n-f+1) x (n-f+1) x nc′`。其中，`nc`为图片通道数目，`nc′`为滤波器组个数。

### Pooling池化层

池化层是CNN中用来减小尺寸、加快运算速度的，还能减小噪声影响，增加健壮性。其不涉及复杂的卷积运算，一般主要有`max pooling`、`average pooling`等。前者即在滤波器滑动区域内取最大值，示意图如下。（超参数p，也就是Padding，在池化层中一般没有）

总之，CNN相比标准神经网络，参数量要少很多，导致训练所需的样本不必很多，从而减小发生过拟合的概率。CNN擅长捕捉区域位置偏移，不易受物体在图片中的位置影响。从而增加物体检测的健壮性。
