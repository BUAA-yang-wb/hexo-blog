## 学习内容
### 向量化
向量化即利用矩阵的思想，大大提高深度学习算法的计算速度。
比如两矩阵相乘，即：
```python
import numpy as np
A = np.random.rand(100)
B = np.random.rand(100)
C = np.dot(A,B)
```

### 逻辑回归——向量化前向传播
训练样本构成的输入矩阵X维度是(n_x,m)，权重矩阵w维度是(n_x,1)，b为常数，输出矩阵Y维度是(1,m)。利用向量化思想，前向传播过程计算如下：
```python
Z = np.dot(w.T,X) + b
A = sigmoid(Z)
```

### 逻辑回归——向量化梯度下降
同理，对于上一篇文章中的梯度下降公式，其向量化为：
```python
dZ = A - Y
db = 1/m * np.sum(dZ)
dw = 1/m * np.dot(X,dZ.T)

w = w - alpha*dw
b = b - alpha*db
```

### 非线性激活函数
常见的非线性激活函数有以下几个：
- sigmoid函数；
- tanh函数；
- ReLU函数；
- Leaky ReLU函数；

注意，绝大部分情况下，隐藏层的激活函数都为非线性，只有输出层的激活函数在一些问题中可以为线性（比如预测问题，输出y为连续的实数范围）。

### 随机初始化
神经网络中的权重矩阵W不能全部初始化为0，原因如下：

> 若W都为0，则隐藏层的多个神经元输出结果相同，涉及到的梯度计算也都一样，导致设置多个神经元无意义。当然，b全部为0是可以的。

一般，可以用以下语句进行初始化：

```python
W_1 = np.random.randn((2,2))*0.01
b_1 = np.zero((2,1))
W_2 = np.random.randn((1,2))*0.01
b_2 = 0
```

乘以0.01的目的是使W一开始比较小，这样当sigmoid或者tanh作为激活函数的话，一开始得到的|z|也比较小，梯度比较大，可以提高梯度下降的更新速度，从而更快的找到全局最优解。
