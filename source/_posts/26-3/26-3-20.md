之前有在vscode中安装过claude code的插件版本，今天想着尝试一下命令行版，看一下这两者有何区别。当然，就目前我的理解来说，插件版+git版本管理+vscode，可以更好地查看代码变更啥的，因为我目前是没办法接受完全脱离代码去“意念编程”的。（claude code模型太贵了，买不起bushi）

回到正文，安装前需要以下前置条件：
- Node.js
- Git
- Powershell
- CCSwitch(方便一键配置不同厂商模型api)

接下来就可以一键安装claude code了（注意，第一次遇到报错了，因为权限不足。可以以管理员身份执行）：

```
npm install -g @anthropic-ai/claude-code
```

因为我之前在vscode中安装过cc插件，在ccswitch中配置过minimax的密钥，所以直接在命令行版中生效了（本质上是帮我自动改写了claude的全局配置文件）。
在powershell中进入要编码的目录，然后就可以开启编码了：
