之前尝试了claude code + minimax2.7 的搭配，只能说一言难尽。就针对当前整个博客网站项目来说，让他修复一个很简单的bug，在那花了十几分钟，消耗了10%的时段额度，最终还理解错了我的需求，一整个无语住了。似乎cc搭配自家模型效果最好，但是太贵了是在消费不起，所以想着尝试一下codex+gpt，以下为具体的流程。

环境属于老生常谈了，要求如下：
- npm >= 10
- node.js >= 22
- win powershell / vscode codex插件

一键安装：`npm install -g @openai/codex`。然后解决token的问题。在浏览相关github仓库时，找到了一个中转站（ccswitch也支持），所以暂且相信它：[PackyCode](https://www.packyapi.com/) 。

登陆网站，注册账号，充值，选择分组codex，其他都是参考的官方文档填写的。令牌导入到ccswitch，就可以开始Coding了。目前就我的感受来说，插件版和命令行版差别不是很大，插件版UI更好。
